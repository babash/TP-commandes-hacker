01 — Démarrer le TP
1
Ouvrir un terminal — cherchez « Terminal » dans les applications, ou appuyez sur Ctrl+Alt+T
2
Télécharger le script — copiez exactement cette commande :
$ wget -O ~/tp_infiltration.sh https://raw.githubusercontent.com/babash/TP-commandes-hacker/main/tp_infiltration.sh
3
Lancer le TP — cette commande initialise tout automatiquement :
$ source ~/tp_infiltration.sh
4
Afficher les questions — tapez ensuite dans le terminal :
$ MISSION
02 — Commandes du TP
MISSION Afficher les 15 questions avec leurs énoncés
Q1 … Q15 Vérifier si la question est réussie — affiche un indice si ce n'est pas le cas
STATUT Voir votre score et progression globale
AGENT Contacter l'agent ECHO pour de l'aide (voir section ci-dessous)

Exemple : vous avez fait la question 3 → tapez Q3 pour savoir si c'est correct.

03 — Contacter l'agent ECHO

Si vous êtes bloqué sur une question, tapez AGENT dans le terminal. Une ligne de communication s'ouvre avec ECHO, votre analyste à distance.

Décrivez votre problème en français : "Je ne comprends pas le pipe", "Quelle est la différence entre ls et cd ?", "Je cherche le PID mais je ne sais pas comment faire"…

ECHO connaît les questions du TP et votre progression. Il vous guidera sans donner la réponse directement. Tapez fin pour fermer la communication.

04 — Les 15 objectifs
Q1pwd
Q2ls
Q3man
Q4ls -la
Q5cd
Q6cat
Q7cat | grep
Q8history | grep
Q9touch
Q10find
Q11mkdir + cp
Q12chmod 600
Q13chmod u+x
Q14top + ps
Q15kill + mv

Les questions en orange introduisent le pipe | — un outil clé du terminal.

05 — En cas de problème
!
Relancer le TP depuis zéro — si quelque chose ne fonctionne plus, retapez simplement :
$ source ~/tp_infiltration.sh
# Tout est réinitialisé : arborescence, espion, progression
