01 Démarrer le TP dans JSLinux
1
Ouvrez JSLinux dans votre navigateur et attendez 15–20 secondes :
https://bellard.org/jslinux/vm.html?url=alpine-x86.cfg&mem=192
2
Téléchargez le fichier tp_infiltration.sh sur votre ordinateur :
https://github.com/babash/TP-commandes-hacker/raw/main/tp_infiltration.sh
Si le navigateur affiche le contenu au lieu de télécharger → clic droit sur le lien → "Enregistrer le lien sous..."
3
Importez le fichier dans JSLinux — cliquez sur la flèche ↑ sous le terminal :
JSLinux — Alpine Linux x86
Welcome to Alpine Linux 3.x
~ # ▌
© Fabrice Bellard ↑  Cliquez ici pour importer un fichier FAQ  |  VM list
Sélectionnez tp_infiltration.sh. Il est copié automatiquement dans ~/.
4
Lancez le TP — tapez dans le terminal JSLinux :
~# . ~/tp_infiltration.sh
# Le point . charge le script (équivalent de "source" — obligatoire sur JSLinux)
⚠ Ne tapez PAS "source" — ce mot n'existe pas sur ce terminal, utilisez le point.
5
Affichez les questions :
~# MISSION
02 Copier-coller dans JSLinux
Ctrl+V ne fonctionne pas. Pour coller du texte copié depuis cette fiche : clic droit sur le terminal → Paste (ou menu du navigateur → Coller).
03 Commandes du TP
MISSIONAfficher les 15 questions avec leurs énoncés
Q1 … Q15Vérifier si la question est réussie — indice précis si échec
STATUTScore et progression globale
AGENTContacter l'agent ECHO pour de l'aide (voir section 04)
Exemple : vous pensez avoir fait la question 7 → tapez Q7 pour vérifier.
04 — Contacter l'agent ECHO

Bloqué sur une question ? Tapez AGENT dans le terminal. ECHO connaît les 15 questions et votre progression — il vous guide sans donner la réponse directement.

Décrivez votre problème en français, puis tapez fin pour fermer la communication.
